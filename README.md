# E-Learning / E-Library System #

 Live Demo:
https://arsh-code03.github.io/E-learning-Arsh/

---

 Overview

This project is a frontend-based E-Library / E-Learning system that I built to understand how a digital library platform works.

The goal was to design a clean and structured interface where users can explore books, view activity, and navigate through different sections like dashboard and issue system.

I focused mainly on UI design, page structure, and basic interactivity using JavaScript.

---

#Features

* Multi-page website (Home, Gallery, Dashboard, Issue)
* Clean and modern dark UI
* Dashboard with recent activity section
* Book browsing layout
* Background animations and smooth effects
* Deployed using GitHub Pages
